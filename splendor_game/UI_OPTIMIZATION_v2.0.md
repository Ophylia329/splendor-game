## 更新时间

## 优化内容

### ✅ 1. 贵族区域优化

#### 位置调整
- **往左移动**: x从1450改为**1300**（左移150px）
- **宽度增加**: 从450增加到**600**
- **更多空间**: 可以容纳多列布局

#### 多列布局
根据贵族数量智能布局：

| 贵族数 | 布局方式 | 说明 |
|--------|----------|------|
| 3个（2人游戏）| 单列垂直 | 传统垂直排列 |
| 4个（3人游戏）| **2×2** | 2列×2行 |
| 5个（4人游戏）| **3+2** | 第1列3个，第2列2个 |

#### 视觉增强
- **贵族尺寸**: 从130增加到**140**
- **需求图标**: 从5px增大到**8px**（增大60%）
- **需求数字**: 使用medium字体（更清晰）
- **间距优化**: 最大间距从13增加到**16**

**布局示意图（4人游戏，5个贵族）：**
```
    贵族
[贵族1] [贵族4]
[贵族2] [贵族5]
[贵族3]
```

## 技术实现

### 贵族多列布局算法
```python
if num_nobles <= 3:
    # 单列：垂直排列
    for i, noble in enumerate(nobles):
        x = base_x
        y = base_y + i * spacing

elif num_nobles == 4:
    # 2×2布局
    for i, noble in enumerate(nobles):
        col = i // 2  # 0或1
        row = i % 2   # 0或1
        x = base_x + col * spacing
        y = base_y + row * spacing

else:  # 5个
    # 3+2布局
    for i, noble in enumerate(nobles):
        if i < 3:
            col, row = 0, i
        else:
            col, row = 1, i - 3
        x = base_x + col * spacing
        y = base_y + row * spacing
```

### 玩家框左右分区
```python
# 左侧信息区（60%）
left_width = int(width * 0.6)

# 右侧保留卡区（40%）
reserved_x = x + left_width

# 左侧垂直排列信息
current_y = y + top_margin
for line in info_lines:
    draw_text(x + left_margin, current_y)
    current_y += line_height

# 右侧垂直排列保留卡
for i, card in enumerate(reserved_cards):
    card_y = y + top_margin + i * (card_height + spacing)
    draw_card(reserved_x, card_y)
```
