# 布局优化调整 v1.1

## 更新时间
2024-12-24

## 问题修复

### ✅ 问题1：宝石列表遮住"公共宝石"标题
**原因**: 宝石列表起始位置 (y=100) 与标题位置 (y=70) 太近

**解决方案**:
- 宝石列表 y 坐标从 100 改为 **130**
- 标题位置从 宝石y-30 改为 宝石y-**40**
- 标题现在显示在 y=90，宝石从 y=130 开始，间隔40px

### ✅ 问题2：玩家区域遮住卡牌
**原因**: 卡牌太大 (140x200)，玩家区域位置 (y=720) 太靠下，卡牌底部被遮挡

**解决方案**:
- **缩小卡牌**: 120x175（从140x200）
- **上移玩家区域**: y=670（从720上移50px）
- **增加玩家区域高度**: 380（从340增加40px，更多空间显示保留卡）
- **减小卡牌间距**: 从15改为12
- **减小等级间距**: 从220改为195

## 新布局参数（基准 1920x1080）

### 区域配置
```python
LAYOUT = {
    'title': {'y': 40},
    'gem_area': {'x': 20, 'y': 130, 'width': 180},     # ⬇️ y+30
    'card_area': {'x': 220, 'y': 100, 'height': 550},  # ⬇️ 高度-50
    'noble_area': {'x': 1450, 'y': 100, 'width': 450},
    'player_area': {'y': 670, 'height': 380}           # ⬆️ y-50, 高度+40
}
```

### 卡牌尺寸
```python
CARD_WIDTH = 120   # ⬇️ -20 (从140)
CARD_HEIGHT = 175  # ⬇️ -25 (从200)
CARD_MARGIN = 12   # ⬇️ -3 (从15)
```

### 间距调整
- **等级间距**: 195px（从220减小）
- **宝石标题距离**: 40px（从30增加）
